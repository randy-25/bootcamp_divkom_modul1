Menjelaskan Sebuah Elemen HTML

Randy Revaldo Pratama - 13222012

div : adalah elemen HTML yang paling general dibandingkan elemen lainnya. Div tidak memiliki efek seperti yang dapat dilakukan oleh elemen b yang untuk bold, hr untuk membuat horizontal line, dan lain - lain. Elemen ini berfungsi sebagai general elemen yang banyak digunakan untuk styling ketika menggunakan CSS dan Javascript untuk membungkus elemen lain agar styling dapat dilakukan dengan tepat.
